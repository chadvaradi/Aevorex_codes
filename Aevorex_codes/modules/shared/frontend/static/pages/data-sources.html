Adatforrások és Harmadik Fél Szolgáltatások

Hatálybalépés dátuma / Utolsó frissítés dátuma: 2024. április 27.

Az Aevorex platform („Szolgáltatások”), beleértve annak minden modulját (pl. FinBot Alpha), elkötelezett az átláthatóság mellett az általunk felhasznált adatok és technológiák tekintetében. A Szolgáltatások funkcionalitásának biztosítása, különösen a pénzügyi adatok megjelenítése, a hírek aggregálása és a mesterséges intelligencia (AI) által generált elemzések nyújtása érdekében különböző, harmadik féltől származó adatforrásokra, Application Programming Interface-ekre (API-kra) és szolgáltatásokra támaszkodunk.

Jelen dokumentum célja, hogy tájékoztatást nyújtson ezen külső forrásokról és szolgáltatókról. Kérjük, vegye figyelembe, hogy ezen harmadik felek saját felhasználási feltételekkel (Terms of Service) és adatvédelmi irányelvekkel (Privacy Policy) rendelkeznek, amelyek szabályozzák az ő szolgáltatásaik használatát és az általuk kezelt adatokat. Az Aevorex nem gyakorol ellenőrzést ezen harmadik felek szolgáltatásai, adatkezelési gyakorlatai vagy az általuk szolgáltatott adatok pontossága, teljessége és időszerűsége felett, és azokért felelősséget nem vállal. Javasoljuk, hogy tekintse át az alább felsorolt szolgáltatók saját dokumentációját.

1. Pénzügyi Piaci Adatok

A Szolgáltatások keretében megjelenített pénzügyi piaci adatok (pl. részvényárfolyamok, historikus adatok [OHLCV], vállalati fundamentumok, pénzügyi mutatók) elsődlegesen az alábbi forrásokból származnak:

  • Yahoo Finance (yfinance library által):
    • Szolgáltatott Adatok: Általános piaci adatok, historikus és (késleltetett) kvázi-valós idejű árfolyamadatok, vállalati alapinformációk, osztalékok, hírek.
    • Szolgáltató Feltételei: A Yahoo Finance saját felhasználási feltételei érvényesek. Az adatok késleltetettek lehetnek és további felhasználásuk korlátozott lehet.
    • Weboldal: https://finance.yahoo.com/
  • Alpha Vantage:
    • Szolgáltatott Adatok: Historikus és valós idejű árfolyamadatok, devizaárfolyamok (FX), kriptovaluta adatok, technikai indikátorok, fundamentális adatok, gazdasági mutatók, hírek és hangulatelemzés.
    • API Kulcs Szükséges: Igen (a `.env` fájlban konfigurálva: `FINBOT_API_KEYS__ALPHA_VANTAGE`)
    • Szolgáltató Feltételei: Kérjük, tanulmányozza az Alpha Vantage felhasználási feltételeit és adatvédelmi irányelveit.
    • Weboldal: https://www.alphavantage.co/
  • Financial Modeling Prep (FMP):
    • Szolgáltatott Adatok: Vállalati fundamentális adatok (mérleg, eredménykimutatás, cash flow), valós idejű és historikus árfolyamadatok, vállalati profilok, pénzügyi mutatók, bennfentes kereskedések, SEC bejelentések, sajtóközlemények, piaci hírek.
    • API Kulcs Szükséges: Igen (a `.env` fájlban konfigurálva: `FINBOT_API_KEYS__FMP`)
    • Szolgáltató Feltételei: Kérjük, tanulmányozza az FMP felhasználási feltételeit és adatvédelmi irányelveit.
    • Weboldal: https://site.financialmodelingprep.com/

2. Hírek és Tartalmi Adatok

A releváns piaci hírek, sajtóközlemények és egyéb szöveges tartalmak gyűjtéséhez és elemzéséhez az alábbi API-kat és forrásokat használjuk (a fentieken túl):

  • MarketAux:
    • Szolgáltatott Adatok: Pénzügyi hírek API, amely híreket, piaci hangulatot és eseményeket szolgáltat globális forrásokból.
    • API Kulcs Szükséges: Igen (a `.env` fájlban konfigurálva: `FINBOT_API_KEYS__MARKETAUX`)
    • Szolgáltató Feltételei: Kérjük, tanulmányozza a MarketAux felhasználási feltételeit és adatvédelmi irányelveit.
    • Weboldal: https://www.marketaux.com/
  • NewsAPI.org:
    • Szolgáltatott Adatok: Globális hírforrásokból származó hírcikkek és főcímek kereshető API-ja.
    • API Kulcs Szükséges: Igen (a `.env` fájlban konfigurálva: `FINBOT_API_KEYS__NEWSAPI`)
    • Szolgáltató Feltételei: Kérjük, tanulmányozza a NewsAPI.org felhasználási feltételeit és adatvédelmi irányelveit.
    • Weboldal: https://newsapi.org/
  • FMP (Financial Modeling Prep) - Hírek:
    • Szolgáltatott Adatok: Az FMP a fundamentális adatok mellett dedikált API végpontokat kínál részvényekkel kapcsolatos hírekhez és sajtóközleményekhez is.
    • API Kulcs Szükséges: Igen (lásd fentebb az FMP szakaszt).
    • Szolgáltató Feltételei: Lásd FMP feltételei fentebb.
  • Alpha Vantage - Hírek és Hangulat:
    • Szolgáltatott Adatok: Az Alpha Vantage is biztosít híreket és piaci hangulatelemzési adatokat API-n keresztül.
    • API Kulcs Szükséges: Igen (lásd fentebb az Alpha Vantage szakaszt).
    • Szolgáltató Feltételei: Lásd Alpha Vantage feltételei fentebb.
  • Yahoo Finance - Hírek:
    • Szolgáltatott Adatok: A Yahoo Finance API (yfinance library) szintén biztosít hozzáférést az adott tickerhez kapcsolódó hírekhez.
    • API Kulcs Szükséges: Általában nem.
    • Szolgáltató Feltételei: Lásd Yahoo Finance feltételei fentebb.

A hírek feldolgozása és rangsorolása a `.env` fájlban meghatározott prioritási sorrend (`FINBOT_NEWS_SOURCE_PRIORITY`) és engedélyezett források (`FINBOT_NEWS_ENABLED_SOURCES`) alapján történik.

3. Mesterséges Intelligencia (AI) Modellek

A Szolgáltatások bizonyos funkciói, mint például az adatok összegzése, a szöveges elemzések generálása vagy a tartalomkészítés támogatása, nagyméretű nyelvi modellekre (LLM) és más AI technológiákra támaszkodnak. Ezekhez a modellekhez jelenleg az alábbi platformon keresztül férünk hozzá:

  • OpenRouter.ai:
    • Szolgáltatás: Egy API aggregátor platform, amely hozzáférést biztosít különböző fejlesztők által készített AI modellekhez egységes interfészen keresztül.
    • Használt Modell (konfiguráció szerint): Jelenleg a(z) `mistralai/mistral-7b-instruct` modellt használjuk (a `.env` fájl `FINBOT_AI__MODEL_NAME` beállítása alapján). Ez változhat.
    • API Kulcs Szükséges: Igen (a `.env` fájlban konfigurálva: `FINBOT_API_KEYS__OPENROUTER`)
    • Fontos Megjegyzés: Az OpenRouter és az általa elérhetővé tett modellek (pl. Mistral AI) saját felhasználási feltételekkel és irányelvekkel rendelkeznek. Az AI által generált kimenetek ("AI Kimenet") pontatlanságokat, hibákat vagy félrevezető információkat tartalmazhatnak ("hallucinációk"). Ezeket az információkat kritikusan kell kezelni, nem helyettesítenek szakmai tanácsadást, és felhasználásuk kizárólag a felhasználó felelősségére történik.
    • Weboldal: https://openrouter.ai/

4. Általános Felelősségkizárás

Az Aevorex törekszik a megbízható és elismert adatforrások és technológiai partnerek kiválasztására. Azonban ismételten hangsúlyozzuk, hogy nem vállalunk felelősséget a harmadik felek által szolgáltatott adatok vagy az AI modellek által generált kimenetek pontosságáért, teljességéért, megbízhatóságáért vagy egy adott célra való alkalmasságáért. A Szolgáltatások használata és az azokon keresztül elérhető információkra való támaszkodás kizárólag az Ön saját kockázatára történik.

További részletekért kérjük, olvassa el a Felhasználási Feltételeinket és a Jogi Nyilatkozatunkat.

5. A Lista Frissítése
